# 🎥 TikTok Live Simulator

Интерактивный веб-симулятор TikTok Live трансляции с реалистичным чатом, донатами и динамичными зрителями.

## ✨ Особенности

- 🎬 **Реалистичный симулятор трансляции** - имитация живой трансляции с анимациями
- 💬 **Интерактивный чат** - напишите сообщение и получите ответы от ботов
- 🎁 **Система донатов** - отправляйте виртуальные донаты с сообщениями
- 👥 **Динамичные зрители** - количество зрителей меняется в реальном времени
- 📊 **Счётчик подписчиков** - следите за растущим количеством подписчиков
- 🎚️ **Уровни сложности** - выбирайте интенсивность активности чата
- 🔊 **Звуковые эффекты** - включайте/отключайте звуки сообщений и донатов
- 📱 **Адаптивный дизайн** - работает на всех устройствах

## 📖 Как использовать

### 💬 Написание в чат
1. Введите сообщение в поле внизу
2. Нажмите кнопку **→** или **Enter**
3. Боты иногда ответят на ваши сообщения

### 🎁 Отправка доната
1. Нажмите кнопку **💎** (Донат)
2. Выберите сумму доната (5₽, 10₽, 50₽, 100₽, 500₽)
3. (Опционально) Напишите сообщение
4. Нажмите **Отправить**
5. Виртуальные донаты появятся на экране трансляции!

### 🎚️ Настройки
- **Сложность** - изменяет частоту сообщений в чате:
  - 🟢 Лёгкая - редкие сообщения
  - 🟡 Средняя - нормальный темп (по умолчанию)
  - 🔴 Сложная - много сообщений
- **Звук** - включение/отключение звуковых эффектов

### Ключевые компоненты

#### 1. **Система зрителей**
```javascript
// Количество зрителей меняется каждые 3-5 секунд
// Диапазон: 18,000 - 39,000
```

#### 2. **Чат-боты**
- 18 различных ботов с реалистичными именами
- 50+ различных фраз
- 10% шанс на подписку с увеличением счётчика

#### 3. **Донаты**
- 5 предустановленных сумм
- Опциональные сообщения к донатам
- Визуальная анимация и звук
- Автоматическое удаление через 5 секунд

#### 4. **Звуки**
- Сообщение: 800Hz beep (0.1s)
- Донат: 1200Hz beep (0.2s)
- На основе Web Audio API

## 🎮 Примеры использования

### Пример 1: Имитация популярной трансляции
1. Установите сложность на **Сложная**
2. Отправьте несколько сообщений в чат
3. Пошлите несколько донатов

## 🔧 Кастомизация

### Изменение диапазона зрителей
В `script.js` найдите:
```javascript
const MIN_VIEWERS = 18000;
const MAX_VIEWERS = 39000;
```

### Добавление новых фраз ботов
```javascript
const botPhrases = [
    'найс',
    'твоя_фраза_здесь',
    // ... остальное
];
```

### Изменение интервалов сообщений
```javascript
const DIFFICULTY = {
    easy: { msgInterval: [3000, 7000], viewerDelta: [100, 300] },
    medium: { msgInterval: [2000, 4000], viewerDelta: [200, 500] },
    hard: { msgInterval: [1000, 2500], viewerDelta: [300, 800] }
};
```

## 🐛 Известные ограничения

- Звуки работают только при первом взаимодействии пользователя с страницей (требование браузера)
- На некоторых мобильных устройствах Web Audio API может быть недоступен
- Чат-боты генерируются случайно, они не "умные" и не читают ваши сообщения
